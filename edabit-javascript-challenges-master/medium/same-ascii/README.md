Return `true`if the sum of ASCII values of the first string is same as the sum of ASCII values of the second string, otherwise return `false`.


### Examples ###
    sameAscii("a", "a") ➞ true

    sameAscii("AA", "B@") ➞ true

    sameAscii("EdAbIt", "EDABIT") ➞ false
