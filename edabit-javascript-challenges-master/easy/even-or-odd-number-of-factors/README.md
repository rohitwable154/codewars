Create a function that returns `"even"`if a number has an even number of factors and `"odd"`if a number has an odd number of factors.


### Examples ###
    factorGroup(33) ➞ "even"

    factorGroup(36) ➞ "odd"

    factorGroup(7) ➞ "even"


### Notes ###
You don't need to actually calculate the factors to solve this problem.
