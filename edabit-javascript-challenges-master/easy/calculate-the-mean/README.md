Create a function that takes an array of numbers and returns the mean value.


### Examples ###
    mean([1, 0, 4, 5, 2, 4, 1, 2, 3, 3, 3]) ➞ 2.54

    mean([2, 3, 2, 3]) ➞ 2.50

    mean([3, 3, 3, 3, 3]) ➞ 3.00


### Notes ###
*   Round to two decimal places.
*   You can expect an integer ranging from 0 to 10000.
