Create a function that returns the number of hashes and pluses in a string.


### Examples ###
    hashPlusCount("###+") ➞ [3, 1]

    hashPlusCount("##+++#") ➞ [3, 3]

    hashPlusCount("#+++#+#++#") ➞ [4, 6]

    hashPlusCount("") ➞ [0, 0]


### Notes ###
It should return `[0, 0]`for an empty string.
