Given N as input, return an array with all primes up to N included.


### Examples ###
    eratosthenes(1) ➞ []

    eratosthenes(10) ➞ [2, 3, 5, 7]

    eratosthenes(20) ➞ [2, 3, 5, 7, 11, 13, 17, 19]

    eratosthenes(0) ➞ []
